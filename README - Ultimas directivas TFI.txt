Trabajo Práctico Final Integrador (Borrador sujeto a modificaciones)

> Fechas Importantes
9/12: Apertura en campus para entrega de proyectos finales.
23/12: Fecha límite de entrega de Proyectos.
30/12: Fecha límite para corrección de Proyectos.
13/01: Descarga de Diplomas: Para que suceda esto, el estudiante debe tener el aprobado en el aula virtual (Tarea del Instructor) y completar la encuesta de satisfacción, de este  modo se va a habilitar el diploma para su descarga.

> Formato de Entrega
Compartir link de Github Pages o Netlify en el apartado de entrega en el Campus Virtual (Requerido)


> Requisitos para la entrega

**Modo Easy**
1. En una sección cargar los items en forma de cards.
2. Tener un botón que muestre la descripción extensa del item.

**Modo Hard**
1. Implementar el Carrito con LocalStorage
2. Consumir una API (de su agrado) y mostrar los datos en una sección.
3. Darle un toque personal. Sugerencias: agregar el botón menos info, cantidad de productos , modal, agregar a la card imagen, usar un fichero json externo, etc.

> Premio
**50.000 $** 

Requisitos para el premio 
Entregar un proyecto con alguna/s características del Hard Mode.
Entregar el 23/12 o antes en el Aula Virtual.
